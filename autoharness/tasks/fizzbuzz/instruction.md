Write a Python script called `fizzbuzz.py` in the current directory.

When run with `python fizzbuzz.py`, it should print FizzBuzz for numbers 1 through 100:
- Print "Fizz" for multiples of 3
- Print "Buzz" for multiples of 5
- Print "FizzBuzz" for multiples of both 3 and 5
- Print the number itself otherwise

Each output on its own line. No extra output.
